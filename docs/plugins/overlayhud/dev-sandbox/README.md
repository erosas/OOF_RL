# Overlay HUD Dev Sandbox

This folder contains reference-only visual sandbox artifacts for Overlay HUD design review.

Files here are not runtime app code.
They are not imported by the WebView frontend.
They are not part of the plugin execution path.
They exist only to preserve visual exploration, mockup comparisons, and design references for future Overlay HUD work.

Do not wire these files into production without creating a scoped implementation task.
